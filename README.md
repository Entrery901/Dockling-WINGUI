# Dockling-WINGUI
**Dockling WINGUI** is an application that **works out of the box** and converts documents (PDF, DOCX, PPTX, XLSX, HTML, etc.) to Markdown for RAG and AI use. People often lack such a ready-made solution; Dockling WINGUI provides it in one program: EXE, folder selection, conversion.  Full guide for installing, configuring, and using the converter.
